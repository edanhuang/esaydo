# EasyDo Logo Pack

Generated from the EasyDo snap-finger logo.

Recommended for Tauri/macOS:
- `icon.icns` / `EasyDo.icns`: macOS app icon bundle.
- `app-icon.png`: 1024x1024 source PNG; can be used with `pnpm tauri icon app-icon.png`.
- `32x32.png`, `128x128.png`, `128x128@2x.png`, `icon.png`: common Tauri desktop icon assets.
- `EasyDo.iconset/`: Apple iconset folder compatible with macOS `iconutil` if you want to regenerate the `.icns` locally.

Note: This is a raster logo generated as PNG, not a true vector SVG. A real SVG would require manual vector tracing/redrawing.
